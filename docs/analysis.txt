Depends upon the saved_model created from
https://github.com/nking/recommender_systems.git

requires python >= 3.10

could use ScANN or FAISS vector search libraries depending upon the
OS.

Chose ScANN because: the set of tools for using tf.Tensors are very
efficient from reading in the data, to creating embeddings with
saved_model signatures, to creating searchable indexes with ScaNN
and storing the translation from insert order to user_id or movie_id
in tf.lookup.StaticHashTables.
The later hash tables could be replaced by cloud services.

ScaNN dependencies:
  - python>=3.9
  - if x86: AVX, FMA instruction set support
  - if ARM: NEON instruction set support
  - if linux: libstdc++ version >= 3.4.23
