goal is to make a Retrieval class for the MovieLens recommender.

the class should be able to:
  - use the two-tower saved model signatures from the
    project https://github.com/nking/recommender_systems.git
    to encode user queries and candidate movie queries
    and get similar recommendations.
  - given a user entry:
    - return similar users
    - return similar movies
    - return whether user is not indexed
  - given a movie entry
    - return similar movies
    - return similar users
  - handle cold starts with weighted Bayesian average.
  - given a user and recommended movies, predict
    the ratings and sort the results for the user.
